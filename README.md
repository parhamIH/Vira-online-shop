# Vira Online Shop

Vira Online Shop is a modern e-commerce platform developed with Django, HTML, CSS, and JavaScript. The project provides a smooth shopping experience with a clean user interface and a scalable backend architecture.

## Features

- User Registration & Authentication
- Product Catalog
- Product Categories
- Shopping Cart
- Responsive Design
- Admin Dashboard
- Product Management
- Secure Django Backend
- Provider management  

## Technologies

- Python
- Django
- HTML5
- CSS3
- JavaScript
- SQLite

## Installation

```bash
git clone https://github.com/parhamIH/Vira-online-shop.git
cd Vira-online-shop

python -m venv venv
source venv/bin/activate

pip install -r requirements.txt

python manage.py migrate
python manage.py runserver
